关于 EasyLearningCS

一个完全离线运行的互动编程教学工具

项目说明

EasyLearningCS 是一个面向编程初学者的互动课程平台。所有课程内容和代码执行均在本地完成， 无需联网。项目使用浏览器端 Python 解释器（Skulpt），让学习者可以直接在浏览器中编写和运行 Python 代码。
